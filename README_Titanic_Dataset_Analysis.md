# 🚢 Titanic Dataset Analysis using Python

This repository contains practice on the Titanic dataset using Python.

## Technologies Used

-   Python
-   Pandas
-   NumPy
-   Jupyter Notebook

## Topics Covered

-   Data loading
-   Missing value handling
-   Data preprocessing
-   Exploratory Data Analysis (EDA)

## How to Run

``` bash
pip install pandas numpy jupyter
jupyter notebook
```
